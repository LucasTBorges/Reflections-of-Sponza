Download main_sponza.zip from https://www.intel.com/content/www/us/en/developer/topic-technology/graphics-research/samples.html
Move the main_sponza.zip file here (/src/assets) and extract the model here.
This should create a new folder containing all necessary files for loading the model: /src/assets/main_sponza.
The new folder contains the model in multiple file formats.
This project uses the gltf model, so the following files can be safely deleted from the folder:
